sfa_load

PURPOSE

SFA_LOAD Load a SFA object.

SYNOPSIS

function hdl=sfa_load(filename)

DESCRIPTION 

 SFA_LOAD Load a SFA object.
   HDL = SFA_LOAD(FILENAME) loads the SFA object saved in the file
   FILENAME and assign to it the handle HDL. The file must have been
   saved by SFA_SAVE.

   See also SFA_SAVE
