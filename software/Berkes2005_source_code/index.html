Matlab source code for the digits recognition simulations performed in (Berkes 2005)

Introduction

This package contains the source code used to produce the results presented in (Berkes 2005).

This program has been written by Pietro Berkes.

Download

The source code requires the MDP Python library.

Download the source code:
.tgz (ca. 6 kB): slowness_model.tgz

The MNIST handwritten digits database formatted as a Python pickle binary is available only on request due to its size (>82 Mb).
